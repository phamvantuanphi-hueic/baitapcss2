CSS selectors

Áp dụng CSS vào các phần khác nhau của HTML

1.Bộ chọn thẻ nhắm mục tiêu vào các phân tử dựa vào tên thẻ HTML của chúng

2.Bộ chọn class nhắm mục tiêu vào các phân tử dựa trên giá trị của thuộc tính class

3.Bộ chọn id nhắm mục tiêu vào các phân tử dựa trên giá trị của thuộc tính id

Bộ chọn thuộc tính nhắm mục tiêu vào các phân tử dựa trên thuộc tính của chúng và giá trị của thuộc tính.

bộ chọn chung nhắm mục tiêu tới tất cả các phân tử.
